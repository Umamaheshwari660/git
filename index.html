Git(Global Information Tracker)

git is a disturbuted version control system that allows multiple developers to collaborate on a project,track chandes to files,and manage project history. it keeps track of all the changes made to files,enabling you to revert to previous versions,collaborate with others,and maintain a clean history of changes.git is widely used in software development for managing source code.

Basic Git Commands

git init:

initialize a new git repository.

git clone:

clone an exisiting repository to your local machine.

git status:

view current status of your working directory and stading area.

git add /Add:

add changes to the staging area before commiting.

git commit -m "message":

commit changes with a message

git push:

push committed changes to a remote repository.

git branch:

list or create branches.

git remote add oridin main:

to link the repository and local file.

git push -u origin main:

to push the local file into remote
